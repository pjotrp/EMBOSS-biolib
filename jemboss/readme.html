Jemboss (v. alpha1.0)

    Jemboss is an interface to EMBOSS.  It is currently being developed at the HGMP-RC, UK. Jemboss can be run in one of two modes. The interface can be run with a local installation, this is the standalone mode. Alternatively, it may be run using a remote server, i.e. in a client-server mode.
